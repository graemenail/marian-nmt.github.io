Detailed Description

A sentence tuple that stores all sources and target sentences for a specific "line" from a parallel corpus.

Sentence tuples are used to store sentences read from external files and to be a basis for construction of marian::data::CorpusBatch objects. They are not a part of marian::data::CorpusBatch.
