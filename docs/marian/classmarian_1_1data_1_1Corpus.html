SentenceTuple marian::data::Corpus::next ( )

Iterates sentence tuples in the corpus.

A sentence tuple is skipped with no warning if any sentence in the tuple (e.g. a source or target) is longer than the maximum allowed sentence length in words.

Returns
A tuple representing parallel sentences.
